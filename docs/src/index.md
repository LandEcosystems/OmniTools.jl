# UtilsKit.jl

A Julia package providing foundational utilities for arrays, collections, display formatting, and type introspection.

## Quick start

```julia
using UtilsKit

# Convert dictionary to NamedTuple
dict = Dict(:a => 1, :b => 2, :c => Dict(:d => 3))
nt = dictToNamedTuple(dict)

# Display a banner
printFIGletBanner("UtilsKit")
```

See the [API](api.md) page for autogenerated docs.

## Submodules (optional)

`UtilsKit` keeps a flat, backward-compatible API (e.g. `dictToNamedTuple(...)`), but the implementation is organized into submodules as well:

```julia
using UtilsKit

UtilsKit.ForArray.positiveMask([1, 0, -1])
UtilsKit.ForNumber.replaceInvalidNumber(NaN, 0.0)
```
